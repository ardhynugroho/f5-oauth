Access Control Demo Guide
====

This is demo guide to configure F5 Access Policy Manager as...

- OAuth Authorization Server, 
- OAuth Resource Server
- Token converter from opaque to JWT

Also to demonstrate user object checking to prevent Broken Object Level Authorization (BOLA) attack
